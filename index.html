About This Page

This HTML page is a simple example to showcase how the CSS variables and base font styles from our discussion can be applied. We're using a sans-serif font family and a responsive font sizing approach where 1rem equals 10px for easier development.

Notice how different elements take on the specified colors and font sizes. This modular approach makes it easy to maintain consistent styling across a large website.
